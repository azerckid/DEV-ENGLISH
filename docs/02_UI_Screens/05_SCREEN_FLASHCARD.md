# Screen Spec: S-06 플래시카드 게임
> Created: 2026-03-08
> Last Updated: 2026-03-08

## 1. 화면 목적

학습한 단어를 카드 앞/뒷면 뒤집기 방식으로 빠르게 점검하는 게임 화면입니다. 맞힘/틀림 선택으로 진행률을 추적합니다.

---

## 2. 컴포넌트 구성

### 2.1 상단 영역
| 컴포넌트 | shadcn/ui | 설명 |
|---|---|---|
| 진행률 바 | `Progress` | 전체 카드 중 완료된 비율 |
| 진행 텍스트 | - | "5 / 20" |
| 맞힘 수 / 틀림 수 | `Badge` | 초록: 맞힘 N개 / 빨강: 틀림 N개 |

### 2.2 플래시카드 영역
| 컴포넌트 | shadcn/ui | 설명 |
|---|---|---|
| 카드 컨테이너 | `Card` | 클릭 시 앞/뒷면 토글 (CSS flip 애니메이션) |

#### 카드 앞면
```
[ 영단어 (word)            ]
[ 예문 빈칸 (_____ 처리)   ]
[ 출처 태그                ]
[ 클릭하여 뒤집기 안내     ]
```

#### 카드 뒷면
```
[ 영단어 (word)            ]
[ 한글 뜻                  ]
[ 예문 전체 (빈칸 채워짐)  ]
[ 출처 태그                ]
```

### 2.3 맞힘/틀림 버튼 (카드 뒷면 공개 후 활성화)
| 컴포넌트 | shadcn/ui | 설명 |
|---|---|---|
| 맞힘 버튼 | `Button` (green variant) | 클릭 시 맞힘 카운트 +1, 다음 카드 |
| 틀림 버튼 | `Button` (red variant) | 클릭 시 틀림 카운트 +1, 다음 카드 |

버튼은 카드 앞면 상태에서 비활성화. 뒤집은 후에만 활성화됩니다.

---

## 3. 상태 정의

| 상태 | 조건 | UI 처리 |
|---|---|---|
| **앞면** | 초기 및 카드 이동 직후 | 단어 + 예문 빈칸 표시. 맞힘/틀림 버튼 비활성화 |
| **뒷면** | 카드 클릭 후 | 뒤집기 애니메이션 실행. 한글 뜻 + 전체 예문 표시. 버튼 활성화 |
| **완료** | 전체 카드 순회 완료 | 결과 요약 화면으로 전환 |
| **Loading** | 단어 목록 fetch 중 | 카드 스켈레톤 표시 |
| **Error** | fetch 실패 | Sonner Toast + 재시도 버튼 |

---

## 4. 결과 요약 화면 (완료 시)

| 컴포넌트 | shadcn/ui | 설명 |
|---|---|---|
| 맞힘 수 / 전체 | - | "15 / 20 맞힘" |
| 정답률 | - | "75%" |
| 틀린 단어 목록 | `Card` | 틀린 단어만 모아서 재확인 |
| 다시 시작 버튼 | `Button` | 전체 카드 셔플 후 재시작 |
| 틀린 단어만 다시 | `Button` (variant: outline) | 틀린 단어만 모아 재시작 |
| 홈으로 버튼 | `Button` (variant: ghost) | S-01로 이동 |

---

## 5. 에러 케이스

| 케이스 | 처리 방식 |
|---|---|
| 단어 목록 없음 | "학습할 단어가 없습니다" + 홈으로 버튼 |
| fetch 실패 | Sonner Toast (error) + 재시도 버튼 |

---

## 6. 반응형

| 구간 | 레이아웃 |
|---|---|
| 데스크톱 (≥768px) | 카드 중앙 정렬, 최대 너비 500px. 맞힘/틀림 버튼 카드 하단 좌우 배치 |
| 모바일 (<768px) | 카드 전체 너비. 버튼 하단 고정 바로 배치하여 엄지 손가락 접근성 확보 |

---

## 7. 카드 순서

- 게임 시작 시 단어 목록을 무작위 셔플 (Fisher-Yates)
- 한 세션 내에서 동일 카드 중복 출현 없음
