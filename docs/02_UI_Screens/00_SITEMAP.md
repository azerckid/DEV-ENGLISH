## 1. 전체 화면 구조

```
DEV-ENGLISH
├── / (단어장 목록)
│   └── /words/[id] (단어 상세 + 어원 분석)
├── /drill (드릴 모드 선택)
│   ├── /drill/fill-in (빈칸 채우기)
│   └── /drill/dictation (받아쓰기)
├── /flashcard (플래시카드 게임)
├── /my-vocabulary (개인 단어장) ← 로그인 필수
├── /contribute (텍스트 기반 단어 기여)
└── /settings (AI API 설정)
```

## 3. 화면별 핵심 구성 요소

### S-01 단어장 목록 (`/`)
- 단어 카드 목록 (영단어 + 한글 뜻 + 예문 출처 태그)
- 검색/필터 (출처 기준: React Docs, MDN 등)
- 각 단어 카드 클릭 시 S-02로 이동
- 상단 네비게이션: 드릴 모드(S-03), 플래시카드(S-06) 진입

### S-02 단어 상세 (`/words/[id]`)
- 영단어 + 발음 기호
- 한글 뜻 및 품사
- 실전 예문 (원문 + 번역 + 출처 링크)
- 어원 분석 섹션
  - 어근/접두사/접미사 분해 및 각 의미
  - 동일 어원 파생어 목록 (개발 용어 위주)
- "내 단어장에 추가" 버튼 (로그인 사용자에게만 활성화)
- 드릴 모드 / 플래시카드 바로가기 버튼

### S-03 드릴 모드 선택 (`/drill`)
- 빈칸 채우기 시작 버튼 → S-04
- 받아쓰기 시작 버튼 → S-05
- 각 모드 간단 설명 텍스트

### S-04 빈칸 채우기 (`/drill/fill-in`)
- 예문 표시 (단어 위치는 빈칸 처리)
- 텍스트 입력 필드
- 제출 후 정답/오답 즉시 피드백
- 진행 상태 표시 (n / 전체)
- 완료 시 결과 요약 화면

### S-05 받아쓰기 (`/drill/dictation`)
- 단어 발음 재생 버튼 (TTS)
- 철자 입력 필드
- 제출 후 정답/오답 즉시 피드백
- 진행 상태 표시 (n / 전체)
- 완료 시 결과 요약 화면

### S-06 플래시카드 게임 (`/flashcard`)
- 카드 앞면: 영어 단어 + 예문 빈칸
- 카드 클릭 시 뒷면: 한글 뜻 + 전체 예문
- 맞힘 / 틀림 버튼
- 진행률 바 (Progress Bar)
- 완료 시 결과 요약 화면

### S-07 텍스트 기반 단어 기여 (`/contribute`)
- 텍스트 붙여넣기 입력창 (공식 문서에서 복사한 내용)
- "단어 추출" 실행 버튼
- 추출된 기술 단어 목록 (체크박스로 선택/해제)
- 선택된 단어에 대한 AI 어원 분석 생성 버튼
- AI 생성 결과 미리보기 (어근/접두사/접미사/파생어)
- "DB에 기여" 확정 버튼
- 기여 완료 시 Sonner Toast 알림

### S-08 AI API 설정 (`/settings`)
- AI 제공자 선택 (Claude API / OpenAI API 라디오 버튼)
- API Key 입력 필드 (마스킹 처리)
- 저장 버튼
- API Key는 브라우저 로컬스토리지에 저장 (서버 전송 없음)

## 4. 글로벌 네비게이션
