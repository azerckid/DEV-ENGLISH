> Created: 2026-03-08

## 1. 주요 사용자 시나리오

MVP 기준 다섯 가지 핵심 사용 패턴을 정의합니다.

---

### Flow A — 단어 학습 후 드릴 진입 (주 학습 경로)

공식 문서를 읽다가 모르는 단어를 찾아보고, 곧바로 드릴로 훈련하는 흐름입니다.

```
[S-01 단어장 목록]
    │
    │ 단어 카드 클릭
    ▼
[S-02 단어 상세]
    │  - 뜻, 예문, 어원 분석 확인
    │
    ├─── "빈칸 채우기 연습" 클릭 ──▶ [S-04 빈칸 채우기]
    │                                      │  ※ 오답 발생 시 → 로그인 사용자면 자동으로 개인 단어장 추가
    │                                      │ 모든 단어 완료
    │                                      ▼
    │                                 [결과 요약]
    │                                      │
    │                               다시 학습 / 홈으로
    │
    └─── "받아쓰기 연습" 클릭 ─────▶ [S-05 받아쓰기]
                                           │  ※ 오답 발생 시 → 로그인 사용자면 자동으로 개인 단어장 추가
                                           │ 모든 단어 완료
                                           ▼
                                      [결과 요약]
                                           │
                                    다시 학습 / 홈으로
```

---

### Flow B — 플래시카드 게임으로 암기 점검 (빠른 복습 경로)

이미 단어장을 둘러본 사용자가 전체 단어를 빠르게 점검하는 흐름입니다.

```
[S-01 단어장 목록]
    │
    │ 네비게이션 "플래시카드" 클릭
    ▼
[S-06 플래시카드 게임]
    │  - 카드 앞면 확인 → 뒤집기 → 맞힘/틀림 선택
    │  - 진행률 바 업데이트
    │  ※ "틀림" 선택 시 → 로그인 사용자면 자동으로 개인 단어장 추가
    │
    │ 모든 카드 완료
    ▼
[결과 요약]
    │
    ├─── 다시 시작 ──▶ [S-06 플래시카드 게임]
    └─── 홈으로 ──────▶ [S-01 단어장 목록]
```

---

### Flow C — 드릴 모드 직접 진입 (반복 훈련 경로)

단어 학습 없이 드릴 훈련만 바로 하고 싶은 사용자의 흐름입니다.

```
[S-01 단어장 목록]
    │
    │ 네비게이션 "드릴 모드" 클릭
    ▼
[S-03 드릴 모드 선택]
    │
    ├─── 빈칸 채우기 ──▶ [S-04 빈칸 채우기] ──▶ [결과 요약]
    └─── 받아쓰기 ──────▶ [S-05 받아쓰기] ────▶ [결과 요약]
```

---

### Flow E — 개인 단어장 관리 (개인화 학습 경로)

로그인 사용자가 개인 단어장을 확인하고 집중 학습하는 흐름입니다.

```
[S-01 단어장 목록 또는 네비게이션]
    │
    │ "내 단어장" 클릭
    ▼
[S-09 개인 단어장]
    │  - 저장된 단어 목록 (수동 추가 / 오답 자동 추가 필터)
    │
    ├─── "드릴 모드로 학습" ──▶ [S-03 드릴 모드 선택] (내 단어장 기준)
    │                                │
    │                                ├─── S-04 빈칸 채우기
    │                                └─── S-05 받아쓰기
    │
    ├─── "플래시카드로 학습" ──▶ [S-06 플래시카드] (내 단어장 기준)
    │
    └─── 단어 개별 삭제 ──▶ [S-09 개인 단어장] (목록 갱신)
```

---

### Flow D — 텍스트 붙여넣기로 단어 기여 (데이터 기여 경로)

공식 문서를 읽다가 기술 단어를 직접 DB에 기여하는 흐름입니다.

```
[S-08 설정] ← API Key 미설정 시 먼저 진입
    │  - AI 제공자 선택 (Claude / OpenAI)
    │  - API Key 입력 및 저장
    │
    ▼ (설정 완료 후)
[S-07 텍스트 기반 단어 기여]
    │
    │ 공식 문서 텍스트 붙여넣기
    │ "단어 추출" 클릭
    ▼
[추출 결과 확인]
    │  - 시스템이 감지한 기술 단어 목록 표시
    │  - 사용자가 기여할 단어 체크박스로 최종 선택
    │
    │ "AI 어원 분석 생성" 클릭
    ▼
[AI 생성 결과 미리보기]
    │  - 어근/접두사/접미사 분해 및 뜻
    │  - 파생어 목록
    │  - 수정 불가 (검수 전 단계)
    │
    │ "DB에 기여" 클릭
    ▼
[기여 완료]
    │  - Sonner Toast: "N개 단어가 미검수 상태로 등록되었습니다"
    │  - 기여한 단어들이 즉시 S-01 단어장에 미검수 배지와 함께 노출
    │
    └─── 홈으로 ──▶ [S-01 단어장 목록]
```
